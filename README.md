# Shipping Box Manager

A simple React app for managing shipping boxes. Add boxes with receiver details, weight, color, and destination country, then view them all in one place. All data is stored locally in your browser.

## What it does

- Add shipping boxes with receiver name, weight, color, and destination
- Calculates shipping costs based on country rates (Sweden, China, Brazil, Australia)
- Stores everything in localStorage - no backend needed
- View all your boxes in a list with their details and costs

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to see it in action.

## Tech Stack

- React 19
- React Router for navigation
- localStorage for data persistence
- Form validation built-in

## Available Scripts

- `npm start` - Run the app in development mode
- `npm build` - Build for production
- `npm test` - Run tests
